# Cahier d’exercices de comptabilité

Application PWA dédiée aux 81 sujets des thèmes 1 à 19 du cahier d’exercices de comptabilité générale.

Principes du projet :
- le PDF source est l’unique référence ;
- aucun exercice inventé ;
- progression enregistrée localement ;
- fonctionnement installable et hors ligne ;
- intégration progressive thème par thème.
